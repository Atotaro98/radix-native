# Blockquote
> Block-level quotation with left border accent.

```tsx
import { Blockquote } from 'radix-native'
<Blockquote>To be or not to be, that is the question.</Blockquote>
<Blockquote size={4} color="blue">Styled quote</Blockquote>
```

Props: `size` (1-9), `weight`, `color`, `highContrast`, plus margin props.
