# Styling

## Token-based props

Components accept design token values instead of raw pixel numbers:

```tsx
// Spacing tokens (0-9) map to: 0, 4, 8, 12, 16, 24, 32, 40, 48, 64 px
<Flex gap={3} p={4}>         {/* gap: 12px, padding: 16px */}
  <Text size={4}>Hello</Text>  {/* font-size: 18px */}
</Flex>
```

## Margin props

Every component accepts margin shorthand props:

```tsx
<Button mt={3} mx={2}>Save</Button>
// marginTop: 12px, marginHorizontal: 8px

<Avatar m={4} />
// margin: 16px all sides
```

| Prop | Maps to |
|------|---------|
| `m` | all margins |
| `mx` | marginLeft + marginRight |
| `my` | marginTop + marginBottom |
| `mt` | marginTop |
| `mr` | marginRight |
| `mb` | marginBottom |
| `ml` | marginLeft |

## Color resolution

Use `useResolveColor` to access theme colors in custom components:

```tsx
import { useResolveColor } from 'radix-native'

function MyComponent() {
  const rc = useResolveColor()

  const bg = rc('accent', 9)        // theme accent at step 9
  const border = rc('gray', 'a6')   // gray alpha step 6
  const text = rc('accent', 'contrast') // text on accent-9 bg

  return <View style={{ backgroundColor: bg, borderColor: border }} />
}
```

## Style prop

All components accept a `style` prop for custom overrides:

```tsx
<Card style={{ maxWidth: 400 }}>
  <Text style={{ textDecorationLine: 'underline' }}>Custom</Text>
</Card>
```
