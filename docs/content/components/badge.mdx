# Badge

```tsx
<Flex direction="row" gap={2}>
  <Badge>Default</Badge>
  <Badge variant="solid" color="green">Active</Badge>
  <Badge variant="outline" color="red">Error</Badge>
</Flex>
```

## API Reference

Inherits from View. Supports [margin props](/theme/spacing).

| Prop | Type | Default |
|------|------|---------|
| `size` | `1 \| 2 \| 3` | `1` |
| `variant` | `'solid' \| 'soft' \| 'surface' \| 'outline'` | `'soft'` |
| `color` | `AccentColor` | theme accent |
| `highContrast` | `boolean` | — |
| `radius` | `RadiusToken` | theme radius |
