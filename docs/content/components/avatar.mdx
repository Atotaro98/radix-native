# Avatar

```tsx
<Flex direction="row" gap={3} align="center">
  <Avatar src="https://i.pravatar.cc/100?img=1" fallback="JD" size={5} />
  <Avatar fallback="A" variant="solid" />
  <Avatar fallback="BG" variant="soft" color="red" />
</Flex>
```

## API Reference

Inherits from View. Supports [margin props](/theme/spacing).

| Prop | Type | Default |
|------|------|---------|
| `src` | `string \| number \| { uri: string }` | — |
| `fallback` | `ReactNode` | **required** |
| `size` | `1-9` | `3` |
| `variant` | `'solid' \| 'soft'` | `'soft'` |
| `color` | `AccentColor` | theme accent |
| `highContrast` | `boolean` | — |
| `radius` | `RadiusToken` | theme radius |

### Fallback

Single-letter fallbacks get a larger font; two-letter fallbacks get a smaller font. All fallback text is auto-uppercased.

```tsx
<Avatar fallback="A" />    {/* larger font */}
<Avatar fallback="JD" />   {/* smaller font */}
```

Non-string fallbacks (icons) automatically receive the correct `color` prop:

```tsx
<Avatar fallback={<PersonIcon />} variant="solid" />
```

### Loading behavior

During image loading, the colored background (placeholder) is shown. Fallback content only appears on error or when no `src` is provided.
