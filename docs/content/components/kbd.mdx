# Kbd

```tsx
<Text>Press <Kbd>Ctrl</Kbd> + <Kbd>S</Kbd> to save.</Text>
```

## API Reference

| Prop | Type | Default |
|------|------|---------|
| `size` | `1-9` | `2` |
| `variant` | `'classic' \| 'soft'` | `'classic'` |

Font-size is 0.75x of the size token. Uses the default body font (not monospace).
